# Film Roll Directory

Drop your best trail photos here. They'll automatically appear on the Film Roll page.

## Workflow

1. **Add photos** to this directory (JPG/JPEG only)
   ```bash
   cp ~/my-ride-photos/best-shot.jpg ./
   ```

2. **Extract EXIF data** (removes GPS/location for privacy)
   ```bash
   ./bin/extract_exif.sh
   ```
   This creates `<photo>_exif.txt` files with camera metadata.

3. **Add captions** (optional)
   Create a text file: `<photo>_caption.txt`
   ```bash
   echo "Emma Long smiles for miles" > best-shot_caption.txt
   ```

4. **Resize for web** (recommended: 1200px max width)
   ```bash
   sips -Z 1200 *.jpg
   ```

5. **Commit and push**
   ```bash
   git add assets/images/film-roll/
   git commit -m "Add film roll photos"
   git push origin main
   ```

## File Naming

- Photos: `anything.jpg` or `anything.jpeg`
- EXIF data: `anything_exif.txt` (auto-generated by script)
- Captions: `anything_caption.txt` (manual, optional)

## What Shows Up

- **Always:** Photographer name (Kenny Gorman), photo, sequential numbering
- **If caption exists:** Your caption text
- **If EXIF exists:** Collapsible "Details" with camera settings (date, camera model, focal length, aperture, ISO, shutter speed, dimensions)

## Privacy

The EXIF extraction script **excludes GPS/location data** automatically.

## Example

```
film-roll/
├── emma-long-smile.jpg
├── emma-long-smile_caption.txt  → "Technical climbing through the rocks"
└── emma-long-smile_exif.txt     → (auto-generated, no GPS)
```
